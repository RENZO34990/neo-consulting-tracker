# Neo Consulting · Sistema de Seguimiento de Requerimientos

Sistema web para gestión y seguimiento de requerimientos de clientes,
desarrollado como prototipo funcional para Neo Consulting.

**Demo en vivo:** https://renzo34990.github.io/neo-consulting-tracker/

## Funcionalidades
- Registro y seguimiento de requerimientos por cliente
- Vista Kanban por estado del flujo
- Dashboard con KPIs operativos
- Alertas automáticas por vencimiento de SLA
- Persistencia local sin backend
